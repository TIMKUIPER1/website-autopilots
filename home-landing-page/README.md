# Autopilots Homepage

Dit is de bewerkmap voor de Autopilots homepage.

## Belangrijkste bestand

- `autopilots-homepage-embed.html`

Dit is de actuele embed-code voor de homepage.

## Gekoppelde GoHighLevel widgets

De pagina gebruikt de originele embeds, zodat bestaande automatiseringen intact blijven:

- Formulier: `EqeJFQsmD1urhHlceuLu`
- Calendar: `UaWTV0sdETiXy0refclQ`

## Connectie-logo's

De integratie-slideshow gebruikt de logo's in:

- `assets/connecties/`
- `assets/connecties-black/`

`assets/connecties/` bevat de originele aangeleverde logo's. `assets/connecties-black/` bevat de gecropte zwarte versies die in de homepage worden gebruikt.

## Samenwerkingslogo's

De samenwerkingen-slideshow gebruikt:

- `assets/samenwerkingen-original/`
- `assets/samenwerkingen-black/`

`assets/samenwerkingen-original/` bevat de aangeleverde webp-logo's. `assets/samenwerkingen-black/` bevat de zwarte gecropte PNG-versies voor de homepage.
