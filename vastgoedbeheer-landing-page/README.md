# Vastgoedbeheer Landingspagina

Dit is de bewerkmap voor de Autopilots landingspagina voor vastgoedbeheerders.

## Belangrijkste bestand

- `autopilots-vastgoedbeheer-landing-embed.html`

Dit is de actuele embed-code die je in GoHighLevel of een andere page builder kunt plakken.

## Gekoppelde widgets

De pagina gebruikt de bestaande embeds, zodat huidige automatiseringen intact blijven:

- Formulier: `EqeJFQsmD1urhHlceuLu`
- Calendar: `UaWTV0sdETiXy0refclQ`
